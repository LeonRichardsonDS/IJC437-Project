# Instructions for repeating the study

### You will need:
 - The employment dataset available in the repository. 
 - The business demography dataset available in the repository. 
 - The four R files available in the repository. 

### First R file: Data processing
 - Before you run this file, fill in the directories in the code so the dataset can be imported off your computer.

### Then run in this order:
 - Data processing
 - RQ1 Area performance
 - RQ2 Industry analysis and PCA
 - RQ3 Linear Regressions

### Note:
 - The libraries, datasets and variables created in previous files may be required in the next.
